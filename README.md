# management-empty

借鉴了一些开源项目的代码和 UI，旨在快速搭建简易 VUE 项目，集成了登陆、注册、路由、状态管理、API 接口请求、忘记密码、返回上一页、UI 主题等功能。

后期可能会新增一些图表、表格、表单、上传的组件，以及 UI 主题切换、路由面包屑等功能。
